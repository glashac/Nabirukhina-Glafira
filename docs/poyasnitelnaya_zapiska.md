## 1. Введение
Кратко опишите предметную область (аренда автомобилей) и цель разработки. Укажите, что целью курсовой работы является реализация клиент-серверного приложения, обеспечивающего учет автомобилей и взаимодействие с пользователем.

## 2. Постановка задачи
Опишите требования, которые вы выполняете:
- Клиент с GUI на JavaFX
- Сервер с REST API (Spring Boot)
- Хранение данных в БД (H2) через ORM (JPA)
- CRUD-операции, фильтрация, сортировка, статистика
- Сохранение/загрузка данных при запуске

## 3. Описание предметной области
Опишите, что такое автопрокат, какие сущности участвуют (автомобиль, заказ, клиент и т.п.). В текущей версии реализована сущность "Автомобиль" (Car).

## 4. Актуальность автоматизации
Обоснуйте пользу автоматизации: удобство хранения данных, быстрота поиска, минимизация ошибок, возможность реализовать отчёты и аналитику.

## 5. Описание программы

### 5.1. Алгоритмические решения
Опишите архитектуру: клиент-сервер, REST API, схема таблицы `cars`, использование JPA.

### 5.2. Описание интерфейса программы
Опишите основные окна:
- Главное окно (таблица автомобилей, кнопки CRUD)
- Диалог добавления/редактирования автомобиля
- Меню "Справка" -> "О программе" / "Об авторе"

### 5.3. Состав приложения
Укажите, какие модули присутствуют:
- Сервер (Spring Boot, REST, H2)
- Клиент (JavaFX, HTTP-клиент)

## 6. Назначение и состав классов программы
Опишите ключевые классы и их назначения (например, `Car`, `CarController`, `CarApi`, `MainApp`, `CarFormDialog`).

## 7. Заключение
Подведите итог, какие требования выполнены и какие возможности реализованы.
